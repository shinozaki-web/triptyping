# 鮨撃 KITAKYUSHU — プロジェクト引き継ぎ文書

## プロジェクト概要

**アプリ名：** 鮨撃 KITAKYUSHU  
**サブタイトル：** 寿司の都タイピング  
**開発者：** 門司港でイタリアンレストランとプログラミング教室を経営するオーナー  
**開発経緯：** 北九州市の「寿司の都構想」とIT教育を掛け合わせた地域密着型タイピングゲーム

---

## 現在の実装状況

### 完成している機能
- 回転寿司レーンを模した流れるタイピングゲームエンジン
  - 見習い（1レーン）/ 板前（2レーン）/ 板長（3レーン）
  - 打ち切った位置で得点倍率変動（左端×3、左半分×2、右半分×1）
  - ターゲット皿固定方式（1つの皿を打ち終えるまで切り替えない）
- タイピングコンテンツ（寿司ネタ・接客フレーズ・北九州観光地・スポンサー枠）
- 10ネタごとのクイズ → パワーアップ付与システム
  - 炙りモード（スコア2倍）/ 冷却タイム（ベルト半速）/ コンボ激化 / おまかせ
- バッジシステム（累計スコア・最高WPM・特別条件、全15種）
- 言語切替（日本語 / バイリンガル / English）
- 北九州方言の音声（正解・ミス時のSpeechSynthesis）
- Web Audio APIによるキー打鍵音・正解音・コンボ音
- プレイヤー登録（ニックネーム・学年・区）
- マイページ（成績履歴・バッジ一覧）
- 一時停止機能（ESCキー対応）
- スポンサー枠（金色グロー演出・バッジ表示）
- ローカルランキング表示

### 既知のバグ（要修正）
1. **クイズ正解→パワーアップ選択後にゲームが再開しない**
   - `skipQuiz()` と `activatePowerup()` の呼び出し順序・状態管理が原因
2. **一時停止後に皿の流れがおかしくなることがある**
   - `cancelAnimationFrame` / `requestAnimationFrame` の二重起動が疑われる
3. **クイズ表示中にタイマーが止まらないことがある**

### 技術スタック（現状）
- 単一HTMLファイル（sushigeki.html）
- Vanilla JS + Web Audio API + SpeechSynthesis API
- データ保存：localStorage（デモ用）
- 外部依存：Google Fonts（Noto Sans JP, Bebas Neue, IBM Plex Mono）のみ

### 収益化の柱（優先順位順）

**① 自治体・学校へのB2B導入（最優先・最大収益）**
- 北九州市教育委員会、各区の小学校への一括導入
- 「寿司の都課」との公式連携で信頼性UP
- 料金目安：学校1校 月額5,000〜30,000円 / 自治体ライセンス 年間50〜200万円
- 提案資料として本アプリのデモが使える状態を早急に整える

**② タイピング大会の参加費・協賛（イベント収益）**
- 区対抗タイピング大会（各区・学年の1位が代表）
- 参加費：個人 ¥500〜1,000
- 協賛：地元企業・飲食店のロゴ掲載＋スポンサーネタとして課題に組み込み
- 大会運営パッケージとして他自治体にも販売可能

**③ スポンサー広告収入（継続収益）**
- タイピング課題にスポンサー企業名が流れる仕組みはすでに実装済み
- 月額掲載料：地元企業 ¥3,000〜10,000/月
- 北九州市内の飲食店・観光施設・不動産などが主なターゲット

**④ 個人・法人サブスク（スケール後）**
- 個人プラン：¥298/月（全モード・成績履歴・広告なし）
- 法人・塾プラン：¥2,980/月〜（生徒管理・成績エクスポート・カスタムモード）

## コンテンツ資産

### タイピング課題（現在約45アイテム）
- 寿司ネタ（lv1：短い名前、lv2：中程度）
- 接客・職人フレーズ（lv2〜3）
- 北九州観光・地名（lv2〜3）
- スポンサー枠（tag付き、金色演出あり）

### クイズ問題（12問）
北九州・寿司をテーマにしたローマ字入力クイズ。正解でパワーアップ獲得。

### 方言フレーズ
- ミス時：「なしか！」「なんしよっと！」「ちゃんとせんか！」など8種
- 正解時：「よかよか！」「さすがばい！」「ばりうまい！」など8種
- ※本番サーバーではVOICEVOX（男性声）に差し替え予定
